# Node.jsの準備（環境構築） 📦

---

### 🐣 「Node.jsってなんですか？ JavaScriptはブラウザで動くって聞いたような...」
### 🦉 「その通り！だが、このNode.js（ノードジェイエス）を入れることで、パソコン本体でもJavaScriptを動かせるようになるんじゃ。『開発のためのエンジン』だと思っておくれ」

---

## 1. Node.jsが入っているか確認する

まず、インストール済みか確認しましょう。

1. **ターミナル（Mac）** または **Git Bash（Windows）** を開きます。
2. 以下のコマンドを入力してEnterキーを押してください。

```bash
node -v
```

- **「v18.xx.x」** のようにバージョン番号が表示されたら、インストール済みです！ → [メインページに戻る](./README.md)
- **「command not found」** や **「認識されていません」** と表示されたら、インストールが必要です。次に進んでください。

---

## 2. Node.jsをインストールする

Node.jsを入れると、便利な道具箱 **「npm」** も一緒にインストールされます。

1. [Node.js公式サイト](https://nodejs.org/ja/) にアクセスします。
2. **「LTS（推奨版）」** と書かれているボタンをクリックしてダウンロードします。
   - ※ 「最新版」ではなく、安定している **「LTS」** を選ぶのがプロのコツです！
3. ダウンロードしたファイルを実行し、インストーラーの指示に従って「Next」を押してインストールします。

インストールが終わったら、もう一度ターミナルを開き直して `node -v` を入力してみてください。バージョンが表示されれば成功です！
