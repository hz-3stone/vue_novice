## 🎭 クラスを付け替える

Vue.jsでは、`:class` を使うことで、条件に応じてクラスを適用できます。
書き方はちょっと変わっていますが、**「条件リスト」** だと思ってください。

```html
<!-- isPushed が true のときだけ '-pushed' クラスがつく -->
<span :class="{ '-pushed': isPushed }">
  {{ telop }}
</span>
```

### この書き方の意味
`{ クラス名: 条件 }` というペアを書きます。これは **「条件リスト」** です。

- **左側**: つけたいクラス名 (`-pushed`)
- **右側**: つけるかどうかの条件 (`isPushed`)

「`isPushed` が正解(true)なら、`-pushed` クラスをつけてね！」という指示になります。

---

## 📝 今回のミッション

### 1. CSSクラスを適用する
`src/App.vue` の `<span class="telop">` を修正して、`isPushed` が `true` のときだけ `-pushed` クラスがつくようにしてください。

```html
<span class="telop" :class="{ '-pushed': isPushed }">{{ telop }}</span>
```

CSSファイル（`style.css`）には、すでに以下のスタイルが用意されています。

```css
/* -pushed クラスがつくと適用される */
.telop.-pushed {
  color: var(--red); /* 赤くなる */
  animation: push 0.5s ease-in-out; /* 揺れるアニメーション */
}
```

### 2. 動作確認
保存してボタンを押してみましょう。
文字が赤くなり、ブルブルと震えれば成功です！

## 🚨 ここでハマる！よくある間違いチェック

### ❌ `class` と `:class` は両方書いていいの？
- **OKです！**
- `class="telop"`（常に適用）と `:class="{...}"`（条件付き）は共存できます。
- Vue.jsが賢いので、両方のクラスを合体させてくれます。

### ❌ アニメーションしない！
- クラス名にハイフンが入っている場合（`-pushed`）、プログラミングのルールの都合上、ちょんちょん `' '` で囲む必要があります。
  - 正解: `{' -pushed': isPushed }`
  - 間違い: `{-pushed: isPushed}` （エラーになります）
