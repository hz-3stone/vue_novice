# Gitの準備（環境構築） 🦑

---

### 🐣 「Gitってなんですか？」
### 🦉 「ゲームのセーブ機能みたいなものじゃ。失敗しても過去の状態に戻せる便利な道具じゃぞ」

---

## 1. Gitが入っているか確認する

まず、あなたのパソコンにGitがすでに入っているか確認しましょう。

1. **ターミナル（Mac）** または **コマンドプロンプト（Windows）** を開きます。
2. 以下のコマンドを入力してEnterキーを押してください。

```bash
git --version
```

- **「git version 2.xx.x」** のように表示されたら、インストール済みです！ → [メインページに戻る](./README.md)
- **「command not found」** や **「認識されていません」** と表示されたら、インストールが必要です。次に進んでください。

---

## 2. Gitをインストールする

### 🍎 Macの方
Macには標準でGitが入っていることが多いですが、最新版を入れるのがおすすめです。
[Git公式サイト](https://git-scm.com/download/mac) からインストーラーをダウンロードして実行してください。

### 🪟 Windowsの方
Windowsの方は、Gitと一緒に **「Git Bash（ギット・バッシュ）」** というソフトも入れるのがおすすめです。これを使うと、Macと同じコマンドが使えるようになって便利です！

1. [Git公式サイト](https://git-scm.com/download/win) にアクセスします。
2. **"64-bit Git for Windows Setup"** をクリックしてダウンロードします。
3. ダウンロードしたファイルを実行します。
4. **重要**: インストール中にたくさんの設定画面が出ますが、**すべて「Next」を押して進めてOKです**。
   - ※ 自動的に「Git Bash」もインストールされます。

インストールが終わったら、**「Git Bash」** というアプリを開いて `git --version` を入力してみてください。バージョンが表示されれば成功です！
