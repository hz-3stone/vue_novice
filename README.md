# Chapter 0: 準備をはじめよう 🛠️

---

### 🐣 新人ヒヨコ「先輩！Vue.jsってやつをやってみたいんですけど、何から始めればいいですか？」
### 🦉 先輩フクロウ「ふむ、よい心がけじゃ。まずは道具を揃えるところからじゃな。料理をするのに包丁とまな板がいるように、プログラミングにも環境が必要じゃ」

---

## 1. 必要な道具を入れる

まずは以下の2つがパソコンに入っているか確認してください。
まだの人は、リンク先を見てインストールしてきてくださいね。

1. **Git (ギット)**: ファイルを保存したりダウンロードする道具
   - [👉 Gitの準備はこちら](./setup_git.md)
2. **Node.js (ノードジェイエス)**: JavaScriptを動かすエンジン
   - [👉 Node.jsの準備はこちら](./setup_node.md)

### 🐣 「両方インストールしました！」
### 🦉 「よし、準備万端じゃな。では早速、この教材のプロジェクトを立ち上げるぞ」

---

## 2. 黒い画面（ターミナル）を開く

ここからはプログラマーっぽく **「ターミナル（黒い画面）」** を使います。
WindowsとMacで開き方が少し違うので注意してください。

### 🍎 Macの方
1. `Command + Space` を押してスポットライト検索を開く
2. 「terminal」と入力して **「ターミナル.app」** を開く

### 🪟 Windowsの方
Gitと一緒にインストールされた **「Git Bash」** を使います。
（これを使うと、Macと同じコマンドが使えるので便利です！）

1. スタートメニューを開く
2. 「Git Bash」と検索して開く
   - または、このフォルダ (`vue_novice`) を右クリックして **「Open Git Bash Here」** を選ぶと、移動の手間が省けて楽チンです！

---

## 3. プロジェクトをダウンロードする（クローン）

### 🐣 「先輩！黒い画面は開きましたが、どうすれば？」
### 🦉 「まずは、レシピ本（ソースコード）を自分のキッチン（パソコン）に取り込むのじゃ」

以下のコマンドを**1行ずつ**コピーして、ターミナルに貼り付けてEnterキーを押してください。

```bash
# 1. GitHubから設計図をダウンロード（クローン）
git clone https://github.com/hz-3stone/vue_novice.git

# 2. ダウンロードしたフォルダの中に入る
cd vue_novice

# 3. 学習用の「learning」ブランチに切り替える
git checkout learning
```

### 🐣 「なんか文字がいっぱい出て止まりました！」
### 🦉 「うむ、エラーが出ていなければ成功じゃ。これでキッチンの準備は整ったぞ」

---

## 4. プロジェクトの準備（コマンド入力）

ターミナルが開いたら、おまじないを入力していきます。
1行ずつコピー＆ペーストして Enter を押してください。

```bash
# 1. 必要なライブラリ（部品）をインストール
npm install

# 2. 開発サーバーを起動！
npm run dev
```

うまくいけば、以下のような文字が表示されます。

```
  VITE v4.x.x  ready in 300 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### 🐣 「なんかURLが出てきました！」
### 🦉 「それが入り口じゃ！ブラウザでそのURLを開いてみるのじゃ」

### ⚠️ 【注意】Windowsの方へ（URLのコピー）
URLをコピーしようとして **`Ctrl + C`** を押すと、サーバーが停止してしまいます！
（ターミナルで `Ctrl + C` は「強制終了」の意味になります）

**URLをコピーしたいときは...**
- 文字を選択して **右クリック → Copy**
- または、URLを **Ctrlキーを押しながらクリック** （これで直接開けます）

もしサーバーが止まってしまったら、もう一度 `npm run dev` と打てば大丈夫じゃよ。

---

## 4. ブラウザで確認する

`http://localhost:5173/` （または表示されたURL）を開いてみてください。
「絶対に押すなよ！？」という画面が出れば成功です！

今はまだボタンを押しても何も起きませんが、これから少しずつ機能を追加して動くようにしていきます。
