[👉 完成デモはこちら（Vercel）](https://vue-novice.vercel.app/)

---

## 🎭 どんなアプリ？

ボタンを押すと、熱湯風呂のリアクション芸のように画面が変化します。
Vue.jsの基本的な機能（イベントハンドリング、条件分岐、データバインディングなど）を使って作られています。

### ✨ 主な機能
- **「PUSH」ボタン**: 押すと何かが起こる...！？
- **リアクション**: 画面の文字と画像が瞬時に切り替わります。
- **リセット機能**: もう一度「フリ」の状態に戻せます。
