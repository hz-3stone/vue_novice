## 📚 この教材の使い方

このリポジトリは、ステップバイステップでVue.jsを学べる構成になっています。
以下の手順で学習を始めてください。

### 1. ローカルで動かしてみる
実際にコードを書きながら進めるために、このリポジトリをあなたのパソコンにダウンロード（クローン）しましょう。

#### コマンドでの手順
ターミナルを開いて、以下のコマンドを入力してください。

```bash
# 1. リポジトリをクローン（ダウンロード）する
git clone https://github.com/hz-3stone/vue_novice.git

# 2. フォルダに移動する
cd vue_novice

# 3. 学習用のブランチに切り替える
git checkout learning
```

✅ **ここでコマンド入力は一旦ストップ！**
このあとの `npm install` や `npm run dev` などの準備は、**Chapter 0** の中で詳しく解説します。
**ターミナルは閉じずに、そのまま開いておいてください！**（閉じるとフォルダの場所がわからなくなってしまいます）

### 2. 学習の進め方
`learning` ブランチの **README.md（Chapter 0）** から読み進めてください。

- **Chapter 0**: 環境構築（ここで `npm install` などをします）
- **Chapter 1〜5**: 手を動かして機能を実装
- **Chapter 6**: 完成＆まとめ

各チャプターの問題（Mission）に取り組み、次のチャプターで答え合わせをする形式です。

### 3. 学習を始める
準備ができたら、以下のリンクから学習用のページ（learningブランチ）へ移動しましょう！

👉 **[学習を始める (learningブランチへ)](https://github.com/hz-3stone/vue_novice/tree/learning)**

## 🆘 トラブルシューティング

### ターミナルを閉じてしまった！ 現在地がわからない！
もしターミナルを閉じてしまったり、コマンドのエラーが出る場合は、フォルダの場所（カレントディレクトリ）が違っている可能性があります。

ターミナルを開き直して、以下のコマンドで `vue_novice` フォルダに入ってください。

```bash
cd vue_novice
```

※ もし `cd: no such file or directory` と出る場合は、保存した場所まで移動する必要があります（例: `cd Documents/vue_novice` など、保存場所に合わせて調整してください）。
