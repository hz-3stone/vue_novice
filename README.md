# Chapter 0: 準備編（キッチンを整える） 🍳

ようこそ、Vue.jsの世界へ！
これから「絶対に押すなよ!?」ボタンアプリを作っていきますが、料理をするにはまずキッチンが必要です。
プログラミングの世界では、これを「環境構築（かんきょうこうちく）」と呼びます。

## このチャプターでやること
- 魔法の呪文 `npm` を使って、必要な道具を揃える
- キッチン（開発サーバー）を立ち上げて、ブラウザに画面を出す

---

## 1. 魔法の道具箱 `npm`

JavaScriptの開発では、先人たちが作ってくれた便利な「道具（ライブラリ）」をたくさん使います。
それを一括で管理してくれるのが **npm**（エヌ・ピー・エム）というツールです。

「カタログ（package.json）」を見て、「道具を全部持ってきて！」とお願いするコマンドがこれです。
ターミナルに打ち込んで、エンターキーをッターン！してください。

```bash
npm install
```

そうすると、文字がたくさん流れてダウンロードが始まります。
終わるまで少し待ちましょう☕

## 2. お店を開店する `npm run dev`

道具が揃ったら、いよいよ開発スタートです。
書いたコードをブラウザで見られるようにするために、「開発サーバー」というものを立ち上げます。
お店のシャッターを開けるイメージですね！

```bash
npm run dev
```

これを打つと、以下のような文字が出てくるはずです。

```
  VITE v5.x.x  ready in 123 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

この `http://localhost:5173/` という部分を、`Ctrl`キー（Macなら`Cmd`キー）を押しながらクリックしてみてください。
ブラウザが立ち上がって、「絶対に押すなよ!?」の画面が出ましたか？

出たら成功です！🎉
この画面が出ているターミナルは、開発中は**そのままずっと開けっ放し**にしておいてください。
（閉じるとお店も閉まっちゃいます！）
