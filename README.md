# Vue.js 実践ハンズオン ♨️

**「HTMLとCSSなら任せて！でもJavaScriptはちょっと…🥺」**
そんなマークアップエンジニアのあなたのための、Vue.js入門コンテンツです！

難しい理屈は一旦置いておいて、「動くモノ」を作りながら楽しく学びましょう！💪

## 🎯 作るもの： 「絶対に押すなよ!?」ボタン

あのおなじみの **「押すなよ？絶対に押すなよ？」** という熱湯風呂ネタをWebアプリで再現します🐣

「押すな」と言われたら「押す」のがお約束。
この美しい様式美を、あなたの手で実装してみませんか？
ボタンを押すと画面が切り替わる、そんなシンプルだけど奥深い体験を作っていきます✨

## 🚀 始め方

今は「完成形」の状態です（ネタバレ注意！🙈）。
学習をスタートするには、ターミナルで以下のコマンドを打ち込んで、学習用モードに切り替えてください！

```bash
git checkout learning
```

切り替えると、このREADMEが「第一章」のテキストに変わります。
そこに書かれているミッションをクリアしながら、少しずつアプリを完成させていきましょう！

## 🎁 このハンズオンで体験すること

- 画面に文字や画像を表示する方法 🖼️
- クリックしたときに「何か」を起こす方法 👆
- 「もし〜なら表示する」という切り替えテクニック 💡
- データの変化に合わせて画面をサラッと変える魔法 ✨

難しい専門用語は使いません！
「あ、こうやれば動くんだ！」という感動を大切に進めていきます。
