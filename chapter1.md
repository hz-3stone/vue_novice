## 🤔 なぜVue.jsを使うの？
HTMLに直接文字を書くと、後で書き換えるのがたいへんです。
Vue.jsを使うと、**「データ（中身）」と「テンプレート（見た目）」** を分けて管理できます。

こうすると、データの中身を変えるだけで、画面上の文字も勝手に書き換わるようになります。

### 🧰 道具箱から取り出す「import」

Vue.jsの便利な機能を使うには、まず「Vue」という道具箱から必要な道具を取り出す必要があります。
それが **`import`（インポート）** です。

```js
// Vueという道具箱から、refという道具を取り出す
import { ref } from 'vue'
```

**【重要】**
この `import` を書かないと、次のステップで使う `ref` が「何それ？」と認識されず、エラーになってしまいます。
「`ref` を使うためには、まず `import` する！」と覚えておいてください。

### 🏷️ 名前をつける「const」

データを作るときは、それに「名前」をつけてあげる必要があります。
**`const`（コンスト）** は、「これはこういう名前のデータですよ」と宣言するものです。

```js
// 「name」という名前の箱を作る
const name = ref('ポチ')
```

### 🪄 魔法の箱「ref」

そして、今回一番大事なのが **`ref`（レフ）** という魔法の箱です。
この箱に入れたデータは、中身が変わると自動的に画面にも反映されます。

1. **道具を取り出す** (`import`)
2. **箱を用意して名前をつける** (`const` + `ref`)
3. **画面に埋め込む** (`template`)

という3ステップです。

### 📝 例：自己紹介アプリのタネ
例えば、名前や年齢を表示する場合こうなります。

```html
<script setup>
import { ref } from 'vue'
import dogImg from './assets/dog.png' // 画像を読み込む

// 1. 魔法の箱（データ）を用意する
const name = ref('ポチ')
const age = ref(3)
const image = ref(dogImg) 
</script>

<template>
  <!-- 2. データを使う -->
  
  <!-- {{ }} は、箱の中身を文字として表示する魔法 -->
  <h1>名前: {{ name }} ({{ age }}歳)</h1>
  
  <!-- : (コロン) は、属性（設定）にデータを接着する魔法 -->
  <img :src="image" />
</template>
```

## 📝 今回のミッション

では、この仕組みを「絶対に押すなよ!?」アプリに組み込んでみましょう！
`src/App.vue` を編集して、以下の3つのデータを作って表示させてください。

### 1. タネ（データ）を作る
`src/App.vue` の `<template>` タグの上に、`<script setup>` を追加して、以下の3つのデータ（ref）を定義してください。

| 変数名      | 初期値               | 説明                        |
| :---------- | :------------------- | :-------------------------- |
| **telop**   | `'絶対に押すなよ!?'` | 上に出るメインの文字        |
| **mainImg** | `furiImg`            | メインの画像 (importが必要) |
| **btnText** | `'PUSH'`             | ボタンの文字                |

※ 画像を使うために、`import furiImg from './assets/furi.png'` も忘れずに！

### 2. タネを仕込む
HTML（template）の中で、直接書いてある文字をデータに置き換えてください。

- 文字の表示には `{{ }}`
- 画像の `src` には `:` (コロン)

**ヒント：**
```html
<span class="telop">ここにデータを埋め込む</span>
<img src="ここにメイン画像を埋め込む" class="main-img" />
<button class="btn">ここにボタンの文字を埋め込む</button>
```

## 🚨 ここでハマる！よくある間違いチェック

### ❌ 画面が真っ白になった！？
- `import { ref } from 'vue'` を書き忘れていませんか？
- `import furiImg ...` のパスは合っていますか？

### ❌ 画像が表示されない！
- `<img src="mainImg">` と書いていませんか？
  - 正解は `<img :src="mainImg">` です。コロン `:` が必要です！
  - コロンがないと、「mainImg」というファイル名を探しにいってしまいます。
