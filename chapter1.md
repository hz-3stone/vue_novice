# 第一章: マジックのタネ（データの定義） 🎩

画面は表示されましたか？
今はまだ、ただのHTMLとCSSで書かれた「絵」です。
これをVue.jsを使って「動かせる状態」にしていきます。

## 🤔 なぜVue.jsを使うの？
HTMLに直接文字を書くと、後で書き換えるのが大変です。
Vue.jsを使うと、**「データ」と「見た目」**を分けて管理できます。

### 📝 例：自己紹介アプリを作るとしたら？
例えば、名前を表示するアプリを作るとします。
Vue.jsでは、まず「箱（変数）」を用意して、そこにデータを入れます。

```html
<script setup>
import { ref } from 'vue'

// 1. データを用意する
const name = ref('ポチ')
const age = ref(3)
</script>

<template>
  <!-- 2. データを使う -->
  <!-- {{ }} で文字を表示 -->
  <h1>私の名前は {{ name }} です</h1>
  
  <!-- : をつけると、属性（設定）にデータを使える -->
  <!-- 例: 画像のパスをデータから設定 -->
  <img :src="'/images/dog.png'" />
</template>
```

こうすると、後で `name` の中身を変えるだけで、画面の文字も勝手に変わるようになります！

---

## 📝 今回のミッション

では、この仕組みを「絶対に押すなよ!?」アプリに使ってみましょう！
`src/App.vue` を編集して、Vue.jsの機能を使って文字と画像を表示させてください。

### 1. タネ（データ）を作る
`src/App.vue` の `<template>` タグの上に、`<script setup>` を追加して、以下の3つのデータ（ref）を定義してください。

- **telop**: 初期値は `'絶対に押すなよ!?'`
- **mainImg**: 初期値は `furiImg` (画像ファイル)
- **btnText**: 初期値は `'PUSH'`

※ 画像を使うために、`import furiImg from './assets/furi.png'` も必要です！

### 2. タネを仕込む
HTML（template）の中で、直接書いてある文字をデータに置き換えてください。
- 文字の表示には `{{ }}`
- 画像の `src` には `:` (コロン)

ヒント：
```html
<span class="telop">ここにデータを埋め込む</span>
<img src="ここにメイン画像を埋め込む" class="main-img" />
<button class="btn">ここにボタンの文字を埋め込む</button>
```
